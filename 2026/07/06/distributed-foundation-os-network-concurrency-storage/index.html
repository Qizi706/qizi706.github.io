协程

协程通常比线程更轻量，适合高并发 I/O 场景。它由用户态调度，不一定每个协程都对应一个内核线程。

对 KV Cache 系统来说，协程适合处理远程读写、元数据查询、异步迁移等 I/O 密集任务。
